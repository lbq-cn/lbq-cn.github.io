# 使用 Echarts 五个基本步骤

  1. 下载 Echarts.js 文件 https://echarts.apache.org/open in new window -〉下载-〉在线定制-〉选择版本下载。

    在网页中引入 Echarts.js 文件。假设下载的文件为 echarts.mini.js，放在 <head> 标签中

    <head>
    <script src="js/echarts.min.js"></script>
</head>

  2. 准备一个具备大小的DOM容器。在 body 中建一个 div，一定要有大小

    <div id="main" 
	style="width: 400px; height: 400px; background-color: green">
</div>

  3. 初始化 echarts 实例对象，用 echarts.init() 初始化

    <script>
	var myEchart = echarts.init(document.getElementById("main"));
</script>

  4. 指定配置项和数据（在 Option 参数中）

    <script>
var option = {
	title: {
		text: "ECharts 入门示例",
	},
	tooltip: {},
	legend: {
		data: ["销量"],
	},
	xAxis: {
		data: ["衬衫", "羊毛衫", "雪纺衫", "裤子", "高跟鞋", "袜子"],
	},
	yAxis: {},
	series: [
		{
			name: "销量",
			type: "bar",
			data: [5, 20, 36, 10, 10, 20],
		},
	],
};
</script>

  5. 将配置项设置给 echarts 实例对象

    <script>
	myChart.setOption(option);
</script>
