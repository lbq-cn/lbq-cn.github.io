# 网络安全现场检查标准

# 1. 外来人员管控

  • 接触办公终端、网络设备和信息机房的外来人员应及时报告至信通分公司

涉及单位大楼、变电站、营业厅所

  • 外来作业人员、日常运维人员必须签订信息保密协议

# 2. 内部人员管理

  • 重要安全岗位人员调动、离职时必须签署保密协议

# 3. 员工网络安全意识

  • 询问员工是否能识别钓鱼邮件及处置过程

不打开陌生内外网邮件，不点击可疑链接，若误点后需立即修改邮箱密码并及时截图反馈至信通分公司

  • 临时离开时应锁定电脑，下班前应关闭电脑

# 4. 信息客服宣传

  • 员工是否知晓本单位信息工作负责人

# 5. 办公终端安全要求

  • 安装桌面管理软件

  • 安装防病毒软件

  • 安装准入软件

  • 桌面管理软件和准入软件的注册人信息是否一致

  • 防病毒软件（360天擎6）程序版本、病毒库升级至最新版本

  • 高危漏洞补丁修复完毕

  • 设置屏幕保护程序

  • 登陆密码为强口令

    长度不小于八位，须字母数字或特殊字符的组合

  • 开启防火墙，封禁高危端口（135-139、445、3389）

  • 禁止在终端显示器、键盘等办公显眼处粘贴电脑及各类信息系统的账号和密码

  • 办公终端应按照使用功能粘贴相应标识标签，明确终端内外网接入情况

  • 严禁外网计算机接入非公司外网网络(例如电信WIFI)

  • 严禁外网计算机存储处理工作秘密

  • 不得使用来历不明的U盘或其他存储介质
