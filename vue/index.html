# 快速入门

# 创建一个 Vue 应用

先想好项目准备放到哪个目录，通过命令行进入该目录，然后执行以下命令

npm init vue@latest # 出现提示，全部默认，一直回车

进入项目目录，安装依赖并启动项目

cd <your-project-name>
npm install
npm run dev # yarn dev

# 清理默认项目页面

清空src\assets 、src\components 目录文件

清空src\App.vue文件内容，仅保留模板基本结构

<!-- App.vue -->
<template>

</template>

# 尝试编写页面内容

在模板标签中直接编写html

<!-- App.vue -->
<template>
  <h1>我的第一个vue页面</h1>
</template>

# 理解vue各项目文件关系

|-- index.html	# 站点首页
|-- public		# 静态资源目录，打包时会放到项目根目录
|-- src
	|-- main.js	# 入口文件
	|-- App.vue	# 根组件
	|-- assets	# 动态资源目录
	|-- components	# 组件库
